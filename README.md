# Unrealipedia — AI-Generated Virtual Creature Encyclopedia

An AI-generated encyclopedia of fictional creatures, created as the media artefact for the
*Generative AI for Media* (Level 7) coursework at Bournemouth University.

Every creature image is generated by a **DDPM** (Denoising Diffusion Probabilistic Model) and
every description is written by a **character-level GPT** — both implemented from scratch in PyTorch.

## Quick start

```bash
# 1. Set up environment (using uv)
uv venv --python 3.12
source .venv/bin/activate
uv pip install torch torchvision --index-url https://download.pytorch.org/whl/cu124
uv pip install -r requirements.txt

# 2. Prepare data
python model1_image_gen/download_data.py --dataset afhq
python model2_text_gen/data/prepare_corpus.py

# 3. Train models
python model1_image_gen/train_ddpm.py --data_root model1_image_gen/data/afhq/train
python model2_text_gen/train.py

# 4. Generate creatures
python pipeline/generate_creatures.py \
    --image_ckpt model1_image_gen/outputs/ddpm/checkpoints/ddpm_epoch_200.pt \
    --text_ckpt model2_text_gen/outputs/gpt/checkpoints/gpt_epoch_100.pt \
    --n 50

# 5. Build website
python website/generate_site.py --creatures pipeline/output/creatures.json
# Open website/dist/index.html in a browser
```

## Models

### Model 1: DDPM (Image Generation)
- **Architecture**: U-Net with residual blocks, self-attention, sinusoidal timestep embeddings
- **Training**: Gaussian diffusion with 1000 timesteps, class-conditional via Classifier-Free Guidance
- **Dataset**: AFHQ v2 (cat, dog, wildlife) or AWA2 (50 animal classes)
- **Output**: 64×64 or 128×128 RGB images of fictional creatures

### Model 2: GPT (Text Generation)
- **Architecture**: Transformer decoder with causal self-attention, ~8M parameters
- **Training**: Character-level next-token prediction on species description corpus
- **Dataset**: Wikipedia animal summaries + fictional creature descriptions
- **Output**: Structured species entries (name, class, habitat, description)
